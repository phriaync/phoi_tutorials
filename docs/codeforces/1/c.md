# [Ancient Berland Circus](https://codeforces.com/problemset/problem/1/C)

显然已经确定的三根柱子一定不共线，于是其确定一个三角形，
所求正多边形一定是该三角形外接圆的一个内接正多边形。

设正多边形每条边所对圆心角为 $t$，则正多边形边数为 $\frac{2\pi}{t}$，
正多边形面积即为 $\pi R^2\sin t / t$。
由于其在题意下随 $t$ 单调递减，我们需要让 $t$ 尽可能大。

设三角形任意两边所对圆心角为 $A, B$，则 $t_{max} = \gcd(A, B, 2\pi - A - B)$。
这是因为所取的角需要覆盖整个圆周，如果是钝角三角形则三根弦所对的圆心角不符合要求。

最后解三角形即可，时间复杂度 $O(1)$。
