# [Tic-tac-toe](https://codeforces.com/problemset/problem/3/C)

细节题。

设场上有 $c_x$ 个 `X` 和 $c_o$ 个 `0`，先判合法。一个局面不合法仅有以下可能：

* $c_x > c_o + 1 \lor c_x < c_o$；
* 双方都胜利；
* `X` 胜利且 $c_x = c_o$；
* `0` 胜利且 $c_x = c_o + 1$。

我们不用考虑一个人赢两次的情况，因为一个人至多能下五颗棋子，
如果这五颗棋子恰好是两个三连，那么其必有一个交点，该玩家最后下在这个交点上即是合法的情况。

然后判某一玩家胜利，再判平局（$c_x + c_o = 9$），最后判下一轮是谁。

该题只有常数。
