# [The least round way](https://codeforces.com/problemset/problem/2/B)

记矩阵为 $\{a_{i, j}\}$，$\alpha_{p}(x)$ 为 $x$ 的质因数分解中质数 $p$ 的幂次。

当矩阵中没有 $0$ 时：

令 $dp_{i, j, 0/1}$ 为走到矩阵第 $i$ 行第 $j$ 列时的 $\min\alpha_{2/5}(x)$，其中 $x$ 为路径上所有数的乘积,则答案为 $\min(dp_{n, n, 0}, dp_{n, n, 1})$。这是因为，$x$ 末尾 $0$ 的个数即为 $\min(\alpha_2(x), \alpha_5(x))$。

于是问题化为朴素的网格 dp，转移方程为：
$$
dp_{i,j,0/1} = \min(dp_{i - 1,j,0/1}, dp_{i,j - 1,0/1}) + \alpha_{2/5}(a_{i, j})
$$

对于路径，可以从终点开始寻找其由哪个方向转移而来。

当矩阵中有 $0$ 时，最后所有数乘积可以为 $0$，于是答案至多为 $1$。因此当 $\min(dp_{n, n, 0}, dp_{n, n, 1}) \ge 1$ 时，将路径条件更改为经过矩阵中至少一个 $0$ 即可。

时间复杂度 $O(n^2)$。
