# [Colorful Intervals](https://atcoder.jp/contests/arc222/tasks/arc222_a)

以左端点为第一关键字、右端点为第二关键字将所有区间排序，使用双指针依次处理每个区间。

考虑用优先队列维护当前可用的最小的数，右指针右移时将数取出并填入数列，左指针右移时使数入队。
如果右指针移动时没有数可用，那么说明需要的最大数大小 $+1$。

这样做必定保证最后数列中最大数最小，因为每次最大数 $+1$ 时都没有数可用，且用更大的数替换已填的数没有意义。

注意需要保证左指针一次连续右移中每个数最多入队一次，最终时间复杂度 $O(N\log N + M\log M)$。

### $O(N + M)$ 做法
记最大区间长度为 $K$，显然答案下界为 $K$。

考虑构造 $1,2,3,...,K,1,2,3,...,K,...$。该构造对于任意一个长度 $\le K$ 的区间，可以保证其中的数两两不同，于是下界 $K$ 能够取到。因此最终最小的最大数为 $K$，构造如上。
