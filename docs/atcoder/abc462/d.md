# [Accomplice](https://atcoder.jp/contests/abc462/tasks/abc462_d)

考虑对每个 $x$ 统计可能罪犯对数。
设有 $c_i$ 个人能在时间 $[i, i + D]$ 犯罪，则从时间 $i$ 开始犯罪的可能罪犯对数为 $c_i(c_i - 1) / 2$ 对。
于是答案为 $\sum_{i \in N} c_i(c_i - 1) / 2$。

考虑维护 $c$，我们发现第 $i$ 个人能在 $[S_i, T_i - D]$ 开始犯罪，这相当于在 $c$ 上进行一次区间加。
而我们只需要在所有区间加完成后得到 $c$， 因此这部分可以差分完成。

最终时间复杂度 $O(N + \max T_i)$。
